# Naming convention

1st_2nd_3rd

## 1st
lp: linear probe
pt: pre-training

## 2nd
1: data 1 % 
100: data 100 %

## 3rd
mb: multiblock
mn: multinoise

## 4th
xy: color mask ratio = xy

so, 'pt100_mn050' means "pre-training multi-noise with color mask ratio = 50"; 'lp100_mn015' means "linear probe for multinoise with color mask ratio = 15 pre-training scheme".
